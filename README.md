# M Beauty Studio - Professional Nail Artist Website

A stunning, modern, interactive website for M Beauty Studio with vibrant colors, 3D effects, and engaging animations.

## 🚀 Quick Deploy

### To Vercel (Recommended - 5 minutes):
1. Push to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "Import" → Select repository → "Deploy"
4. Done! ✅

### To Netlify:
1. Go to [netlify.com](https://netlify.com)
2. Connect GitHub repository
3. Auto-deployed! ✅

### Local Preview:
Double-click `index.html` to open in your browser
